# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities by opening a [GitHub Security Advisory](https://github.com/WokSpec/WokTool/security/advisories/new).

Do **not** open a public issue for security vulnerabilities.

We will respond within 72 hours and aim to patch confirmed vulnerabilities within 7 days.

## Scope

- XSS vulnerabilities in tool implementations
- Client-side data exposure
- Dependency vulnerabilities with active exploits

## Out of Scope

- Theoretical vulnerabilities with no practical exploit
- Issues in tools that explicitly require external API keys (those are the user's responsibility)
